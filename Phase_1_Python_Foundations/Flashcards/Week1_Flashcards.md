# Flashcards: Week 1 Review

## Day 1: Variables & Print

| Question | Answer |
|----------|--------|
| How do you create a variable? | `name = "John"` (no $) |
| How to print with variable? | `print(f"Hello {name}")` |
| Multiple assignment? | `x, y = 1, 2` |
| Check variable type? | `type(var)` |

---

## Day 2: Data Types

| Question | Answer |
|----------|--------|
| Integer? | `42` |
| Float? | `3.14` |
| Boolean? | `True` / `False` |
| None equivalent? | `None` (not null) |
| String to int? | `int("42")` |
| Int to string? | `str(42)` |
| 7.9 to int? | `7` (truncates!) |

---

## Day 3: Strings

| Question | Answer |
|----------|--------|
| F-string syntax? | `f"Hello {name}"` |
| String length? | `len(text)` |
| Get char at index 3? | `text[3]` |
| Slice 0-3? | `text[0:3]` or `text[:3]` |
| Reverse string? | `text[::-1]` |

---

## Day 4: String Methods

| Question | Answer |
|----------|--------|
| Lowercase? | `text.lower()` |
| Find position? | `text.find("abc")` or `text.index("abc")` |
| Replace text? | `text.replace("old", "new")` |
| Split by comma? | `text.split(",")` |
| Join list? | `" ".join(list)` |
| Strip whitespace? | `text.strip()` |

---

## Day 5: Lists

| Question | Answer |
|----------|--------|
| Create list? | `[1, 2, 3]` |
| First element? | `list[0]` |
| Last element? | `list[-1]` |
| Length? | `len(list)` |
| Check membership? | `item in list` |
| Get 2nd to 4th? | `list[1:4]` |

---

## Day 6: List Operations

| Question | Answer |
|----------|--------|
| Add to end? | `list.append(item)` |
| Add to index 1? | `list.insert(1, item)` |
| Remove last? | `list.pop()` |
| Remove by value? | `list.remove(item)` |
| Sort in place? | `list.sort()` |
| Sorted copy? | `sorted(list)` |
| List comprehension? | `[x*2 for x in list]` |

---

## Day 7: Input & Casting

| Question | Answer |
|----------|--------|
| Get input? | `input("Prompt: ")` |
| Input returns? | String (always!) |
| Convert to int? | `int(var)` |
| Convert to float? | `float(var)` |
| Safe int input? | `try: int(input()) except ValueError` |

---

# Daily Review Routine (15 minutes)

**Morning (5 min):** Read these flashcards  
**Evening (10 min):** Re-type Day 1 exercises from memory  

**Repeat:** Review Day 1 flashcards on Day 3, Day 7, Day 14
