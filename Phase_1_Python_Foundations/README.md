## Quick Reference: PHP → Python

```python
# PHP                          # Python
$name = "John";               name = "John"
$age = 25;                    age = 25
$is_active = true;            is_active = True
$prices = [1, 2, 3];          prices = [1, 2, 3]
$prices[] = 4;                prices.append(4)
echo "Hello $name";           print(f"Hello {name}")
count($arr);                  len(arr)
array_push($arr, 4);          arr.append(4)
array_keys($dict);            dict.keys()
in_array($val, $arr);         val in arr
```
